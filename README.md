# README #

Cardio Explorer UI

### What is this repository for? ###

* This repository contains the code for the Exploris UI for Cardio Explorer
* Version 0.0.1

### How do I get set up? ###

* Clone this repository
* Install Node.js and npm
* Run npm install
* Run npm run dev

### How do I run the tests? ###
* Run npm run test
